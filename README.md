Music Recommendation System

A content-based music recommendation system built using the Spotify Web API and Python. The system connects to Spotify using the Spotipy library and extracts rich audio features and metadata from playlists, including track popularity, artist genres, danceability, energy, tempo, and other audio characteristics provided by Spotify's audio analysis engine.

These features are fed into a recommendation model that identifies and suggests musically similar tracks based on the properties of a given song. The project includes a web frontend built in HTML and CSS that allows users to interact with the system and browse recommendations in the browser.

To run the project, replace the Client ID and Client Secret in extraction.py with your own Spotify Developer credentials from developer.spotify.com, then launch the recommendation app locally.
